# SSTCE: A Solution Space Transformation-Guided Co-Evolution for Energy-Saving Distributed Heterogeneous Flexible Job Shop Scheduling

DATASET: All datasets are included in this folder.

CriticalPath.py: Find the critical path.

DataRead.py: Data preprocessing.

EA.py: Evolutionary operators.

EnergySave.py: Energy save strategy.

FastNDSort.py: Fast non dominated sorting.

LocalSearch.py: Local search operators.

fitFJSP.py: Calculate fitness.

inital.py: The hierarchical clustering-based population initialization strategy.

parameter.txt: Parameter document, including ps, Pc, Pm, epsilon, and theta.
